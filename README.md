# 🌐 Language Translation Tool

A Streamlit-based web app that translates text or voice input between multiple languages, with text-to-speech playback of the translated result.

## Features
- Translate typed text between 100+ languages using Google Translate
- Voice input with multi-language speech recognition (English, Telugu, Hindi, etc.)
- Text-to-speech playback of translated output
- Simple, clean UI built with Streamlit

## Tech Stack
- Python
- Streamlit
- deep-translator
- SpeechRecognition
- gTTS (Google Text-to-Speech)

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
